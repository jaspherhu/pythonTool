README - 时代换电代码转换器
一、功能概览
本工具是基于Python Tkinter开发的代码转换工具，支持以下核心功能：

1. 核心转换功能

关键词替换
根据keyword_config.csv配置文件进行整词替换
支持自动生成唯一映射值
支持正则表达式整词匹配替换
字符映射
随机生成A-Z/a-z/0-9的字符映射表
自动保存反向映射表reverse_mapping.json
2. 恢复功能

支持通过反向映射表恢复原始内容
自动检测并处理映射冲突
3. 辅助功能

文件编码自动检测（支持UTF-8/GBK/GB2312等）
扩展名过滤（默认处理.c/.h文件）
实时操作日志记录
可视化进度条显示
映射表/关键词管理界面
4. 安全机制

配置文件自动备份
冲突检测与自动修复
空映射值自动补全
二、使用说明
1. 首次运行准备

bash
├── mapping_config.json       # 自动生成的字符映射表
├── keyword_config.csv        # 关键词映射配置文件
└── reverse_mapping.json      # 自动生成的逆向映射表
2. 界面操作流程
UI示意图

步骤1：选择目标目录

点击浏览...按钮选择代码目录
在日志区确认目录路径
步骤2：配置参数

python
# 文件扩展名（默认：.c, .h）
输入框示例：.c, .h, .cpp

# 文件编码（默认gb2312）
支持编码：utf-8 / gbk / big5 等
步骤3：执行转换

功能按钮	说明
开始关键词转换	执行CSV配置的整词替换
开始字符映射	执行随机字符映射
恢复关键词	根据历史配置还原关键词
恢复字符映射	通过反向映射表还原字符
步骤4：监控进度

进度条显示处理进度
日志区实时显示处理详情
三、高级功能
1. 关键词管理

点击关键词按钮打开管理窗口
支持功能：
添加/删除关键词映射
自动生成唯一映射值
映射冲突可视化提示
CSV配置导入导出
2. 映射表管理

点击映射表按钮查看字符映射关系
点击刷新映射表生成新随机映射
四、注意事项
文件备份建议
执行操作前建议备份原始代码

编码兼容性
当出现乱码时尝试切换文件编码设置

冲突处理

重复关键词自动覆盖
映射值冲突时自动生成新值
日志追踪
完整操作记录保存在operation.log

五、运行环境
系统支持：Windows/macOS/Linux
Python版本：3.6+
依赖库：tkinter, chardet, requests
安装依赖：

bash
pip install chardet
